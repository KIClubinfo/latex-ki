# KI LaTeX themes

## Description

- `enpc-ki` : basic theme
- `enpc-ki-beamer-theme` : beamer theme for awesome presentations

## Installation

(not tested for beamer theme)
``` bash
make install
```
or (only written for enpc-ki)
``` bash
./install.sh
```

What it does :
* put ttf fonts in ~/texmf/fonts/truetype
* put tex files and images anywhere in ~/texmf
